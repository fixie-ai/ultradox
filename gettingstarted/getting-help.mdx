## What to Include When Asking for Help

To help us assist you more effectively, please include:

1. **Description**: clear description of what you're trying to achieve
1. **Code snippets**: relevant code or configuration details (remove any API keys!)
1. **Error messages**: any errors or unexpected behavior you're experiencing
1. **Steps to reproduce**: details of the steps used to reproduce the issue (if applicable)
